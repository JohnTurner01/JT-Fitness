☁️ Cloud Sync — Supabase (Optional)

Enable this to sync your data across multiple devices (phone, laptop, etc.). Uses Supabase — free for personal use. Leave blank to keep using local storage only.

How to set this up (free, ~5 mins):
1. Go to supabase.com and create a free account
2. Create a new project (pick any name and region)
3. Go to Project Settings → API
4. Copy your Project URL and anon/public key into the fields below
5. Go to SQL Editor and run the setup query shown below
create table if not exists apex_health_data ( id uuid default gen_random_uuid() primary key, user_id text not null, data jsonb not null, updated_at timestamptz default now() ); alter table apex_health_data enable row level security;

Data Management

All data is stored locally in your browser's localStorage. Nothing leaves your device unless you explicitly use AI features or enable Cloud Sync above.
